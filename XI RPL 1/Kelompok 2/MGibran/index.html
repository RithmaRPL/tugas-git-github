Saya berperan dalam pembuatan slide presentasi, khususnya pada bagian Learning Objectives dan Topik. Selain itu, saya juga menyusun dan memaparkan materi mengenai Git Melihat Detail perubahan

git diff

Untuk melihat secera jelas perubahan yang terjadi Seperti baris mana yang ditambah (+) atau dihapus (-)

git status

Untuk menampilkan status file di repository seperti : File baru (untracked), File yang berubah tapi belum di-stagge, dan File yang sudah siap di-commit

saat sudah di add akan berubah menjadi warna hijau seperti ini yang artinya sudah siap di commit

git log

Berfungsi untuk melihat riwayat commit secara lengkap Seperti siapa yang commit, kapan, dan pesan commit

git log oneline

Menampilkan riwayat commit secara singkat (1 baris per commit) Supaya Lebih mudah dibaca dan untuk melihat daftar commit
